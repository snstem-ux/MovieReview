# 영화 일지 (Movie Journal)

## 1. 한 줄 정의

내가 본 영화를 기록하고 별점이랑 한 줄 감상을 남기는 나만의 영화 일기.

## 2. 사용자 시나리오

1. 방금 본 영화를 30초 안에 기록한다
2. 최근 본 영화 목록을 한눈에 본다
3. 별점이나 태그로 필터링해서 본다

## 3. 할 것 / 안 할 것

- **꼭 한다**: 영화 추가/수정/삭제, 별점, 한 줄 감상, 목록 보기
- **하면 좋다**: 태그, 검색
- **여유되면**: 포스터 이미지, 통계
- **안 한다**: 회원가입, 댓글, SNS 공유, 다른 사람과 공유

## 4. 기술 스택

- Next.js (App Router, TypeScript, LTS 버전)
- Tailwind CSS
- 라이브러리 설치: 모든 종속성은 LTS 버전을 기본으로 설치
- 데이터는 브라우저 localStorage에 저장 (서버/DB 없음)
- Vercel로 배포

## 5. 저장할 정보

영화 한 편당 다음 정보를 기록합니다:

- 영화 제목
- 본 날짜
- 별점 (1~5)
- 한 줄 감상
- 태그 (예: 'SF', '재관람')

## 6. 화면 구성

- 메인: 영화 목록 + 우상단 "추가" 버튼
- 추가: 새 영화 입력 폼
- 상세: 영화 클릭 시 자세히 보기 / 수정 / 삭제

## 7. 디자인 분위기

참고 사이트는 Linear (linear.app). 이런 느낌으로:

- 다크 모드만 사용 (배경 거의 검은색, 글씨는 살짝 회색빛 흰색)
- 포인트 컬러 한 가지만 (연한 보라 정도)
- 여백 넉넉히, 깔끔하게
- 테두리는 거의 안 보이게 얇게
- 카드 모서리는 살짝만 둥글게
- 한글은 Pretendard, 영문은 시스템 폰트
- 애니메이션은 빠르고 부드럽게
- 데스크톱 우선, 모바일은 동작만 되면 됨

## 8. 추가 제약

- 외부 API 호출 안 함
- 이미지 업로드 안 함 (포스터는 URL만 받음)
- 한국어만 지원
